# Base de Datos - QUIMIKARDEX

Esta carpeta contiene todos los scripts SQL necesarios para reconstruir completamente la base de datos del proyecto en Supabase.

## Contenido

- Creación de tablas
- Relaciones
- Constraints
- Índices
- Enum Types
- Funciones RPC
- Triggers
- Row Level Security (RLS)
- Policies
- Configuración de Storage

## Instalación

1. Crear un proyecto en Supabase.
2. Abrir SQL Editor.
3. Ejecutar los scripts en el orden correspondiente.
4. Crear el bucket de Storage.
5. Configurar las políticas.

Después de ejecutar todos los scripts, la aplicación estará completamente funcional.
